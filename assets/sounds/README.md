# Ses Dosyaları

Bu klasöre aşağıdaki ses dosyalarını ekleyin:

## Gerekli Ses Dosyaları:

### 1. success.mp3
- **Açıklama**: Başarılı işlemler için
- **Kullanım**: Tema seçimi, tooltip tamamlama
- **Süre**: 1-2 saniye
- **Ton**: Pozitif, yukarı doğru

### 2. tap.mp3
- **Açıklama**: Buton tıklamaları için
- **Kullanım**: Navigasyon, skip butonları
- **Süre**: 0.5 saniye
- **Ton**: Hafif, net

### 3. confetti.mp3
- **Açıklama**: Kutlama efektleri için
- **Kullanım**: Konfeti animasyonu
- **Süre**: 2-3 saniye
- **Ton**: Neşeli, çok sesli

### 4. notification.mp3
- **Açıklama**: Bildirim sesleri için
- **Kullanım**: Genel bildirimler
- **Süre**: 1 saniye
- **Ton**: Dikkat çekici, yumuşak

## Ses Dosyası Kaynakları:
- [Zapsplat](https://www.zapsplat.com/) - Ücretsiz ses efektleri
- [Freesound](https://freesound.org/) - Açık kaynak sesler
- [Adobe Audition](https://www.adobe.com/products/audition.html) - Ses düzenleme

## Format Önerileri:
- **Format**: MP3
- **Bitrate**: 128 kbps
- **Sample Rate**: 44.1 kHz
- **Kanal**: Mono veya Stereo
- **Dosya Boyutu**: < 100KB (performans için)
