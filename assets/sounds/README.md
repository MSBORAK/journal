# Bildirim Sesleri

Bu klasör bildirim seslerini içerir.

## Ses Dosyaları:

- `gentle.mp3` - Nazik hatırlatmalar için yumuşak ses
- `alert.mp3` - Görev hatırlatmaları için dikkat çekici ses  
- `celebration.mp3` - Başarı bildirimleri için kutlama sesi

## Not:
Şu anda placeholder ses dosyaları kullanılıyor. Gerçek ses dosyaları eklendikçe bu dosyalar güncellenecek.

## Ses Dosyası Formatı:
- Format: MP3
- Süre: 1-3 saniye
- Kalite: 128kbps veya daha yüksek
- Kanal: Mono veya Stereo
