# Background Images

Bu klasöre aşağıdaki görselleri ekleyin:

## Gerekli Dosyalar:

### 1. galaxy.jpg
- **Milky Way** fotoğrafı
- **Yıldızlı gökyüzü** 
- **Koyu mavi/siyah** tonlar
- **Dikey format** (portrait)
- **Yüksek çözünürlük** (minimum 1080x1920)

### 2. aurora.jpg  
- **Aurora Borealis** (Kuzey Işıkları)
- **Yeşil/mor/teal** tonlar
- **Dinamik** görünüm
- **Dikey format**
- **Yüksek çözünürlük**

### 3. sunset.jpg
- **Gün batımı** manzarası
- **Turuncu/sarı/kırmızı** tonlar
- **Sakin** atmosfer
- **Dikey format**
- **Yüksek çözünürlük**

## Görsel Kaynakları:

### Ücretsiz Kaynaklar:
- **Unsplash.com** - Profesyonel fotoğraflar
- **Pexels.com** - Yüksek kalite
- **Pixabay.com** - Telif hakkı yok

### Arama Terimleri:
- "milky way night sky"
- "aurora borealis northern lights"
- "sunset over water"
- "galaxy stars"
- "night sky astrophotography"

## Dosya Formatları:
- **JPEG** formatında
- **Optimize edilmiş** boyut (2-5MB max)
- **Progressive JPEG** önerilen
